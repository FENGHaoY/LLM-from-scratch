## 📖 项目简介

本仓库是基于 [skindhu/Build-A-Large-Language-Model-CN](https://github.com/skindhu/Build-A-Large-Language-Model-CN) 电子书的**代码实现**。该书深入探讨了大语言模型（如 GPT）的原理与实现细节，是学习 LLM 架构、训练过程及应用开发的优秀教材。

**本项目的特色**：
- ✅ **逐层实现**：从 Token 化到 Transformer，每个模块都有详细代码
- ✅ **思想整合**：将书中的理论、方法、代码进行系统化整合
- ✅ **从零到一**：不依赖高级封装，帮助你真正理解 LLM 内部机制
- ✅ **可运行代码**：所有代码均经过测试，可直接运行学习
